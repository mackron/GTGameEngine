You probably already have most prerequisite packages already installed. You will need to install packages for the following libraries:

  • PCRE
  • FreeType 2
  • libpng/zlib
  • OpenAL

On Arch Linux, use the following command to install the prerequisite packages.

sudo pacman -S pcre freetype2 libpng zlib openal

Please note that Linux is not my primary platform, so there may be some annoyances and bugs. It would be great if you could send me a quick tweet or email and let me know if you find a problem.
